## windows下scrapy安装

### 下载依赖的第三方离线包
到 [这个地址](https://www.lfd.uci.edu/~gohlke/pythonlibs/#twisted)下载一些在windows平台编译的第三方库.
1. pywin32
2. twisted
**注意:** 根据python的版本下载对应的离线包.

### 安装或更新依赖的第三方离线包
下载好后就安装
1. 在控制台切换目录到离线包所在的目录
2. 安装离线包
```
pip install pywin32
pip install twisted
```
如果是更新包则:
```
pip install pywin32 -U
pip install twisted -U
```

### 安装或更新scrapy
下载安装
```
pip install scrapy
```
下载更新
```
pip install scrapy -U
```
