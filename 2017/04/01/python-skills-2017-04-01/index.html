1. windows下Python2与Python3共存的使用方法

windows下同时安装了Python2.7.13与Python3.6后,默认的使用python2.

  • 调用python2,使用 py -2
  • 调用python3,使用 py -3
  1. 使用pip安装库
  • 安装到Python2时，就使用 pip2 install [name]
