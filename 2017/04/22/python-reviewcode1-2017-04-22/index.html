2. pwd,$pwd与cwd

都指某个进程运行时所在的目录.
pwd 是linux 自带的命令.全称:pathname of the current working directory.
$PWD 是个系统变量
cwd: 不是系统自带的命令, 但是属于系统的属性.全称: current working directory . 不但在 /proc/{id} 这个目录下可以看到cwd, 在很多其他的编程语言中也可以看到( 例如grunt ).
有时候 pwd 与 $PWD 给出的结果不同.
